# Book Nest - App

## Technology used:

- Next.js
- MantineUI
- TypeScript
- React-Query
- Pagination and Filtering
- Real time notification

## Instructions to run the app

- pnpm/npm/yarn install
- pnpm/yarn dev || npm run dev
